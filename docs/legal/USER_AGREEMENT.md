# 发票管家用户使用协议
更新日期：2026-05-10

## 1. 使用范围
- 本工具用于发票识别、归档、报销、统计、导出与验真等办公处理场景。
- 你应仅处理自己有权访问、保存或提交的票据与相关业务数据。
- 本工具不替代法定票据保管义务，重要数据请自行备份。

## 2. 用户责任
- 你应自行复核 OCR、验真、报销汇总与导出结果，避免将未复核数据直接用于正式报销、入账或审计材料。
- 你不得利用本工具处理违法、侵权、伪造或与业务无关的票据数据。
- 你应妥善保管 API 验真、RPA 验证码识别等外部服务凭据。

## 3. 外部服务与限制
- 启用 API 验真或 RPA 验真时，程序会按所选模式向外部服务发送必要数据。
- RPA 验真依赖自动化浏览器与第三方验证码识别服务，可能受网络、平台策略或服务额度限制影响。
- 如外部服务不可用、限流或返回异常，本工具不保证查验一定成功。

## 4. 数据与合规
- 本工具默认将业务数据保存在本地目录中，具体范围与外部传输触发条件以《隐私与数据处理说明》为准。
- 本工具默认仅处理发票识别、验真、归档、报销、导出所必需的数据，不用于采集与业务无关的个人信息。
- 如你所在组织对发票、报销或个人信息处理有额外制度要求，你应先满足组织内部合规要求后再使用本工具。
- 你应确保对所处理数据的收集、存储、传输和导出行为符合适用法律法规。

## 5. 版本与更新
- 当隐私说明、协议条款或关键外部服务依赖发生实质变化时，应用可能要求重新确认后方可继续使用。
- 继续使用新版本应用，即表示你已阅读并接受最新条款。
